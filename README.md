# [MinecraftMod] DawnBreaker [MC1.8.9]
##■DL(v2.0)
https://www.dropbox.com/s/y490if6tos35mwl/DB_v2_M189_F1671.jar?dl=0

##■はじめに  
- このModは「TES V:Skyrim」に登場する剣「Dawnbreaker」の模造品(以下、DB)を追加します。
- 若干、本家とは異なる部分があります。(レプリカ、という事でひとつ)
- 予告無しに仕様変更を行う事があります。
- 他Modによる競合・不整合は関知しません。
- 作りが荒い為、不具合等が発生する恐れがあります。
- 定期的なバックアップが取れて、不具合に直面しても泣かない強い子が使ってください。

##■特性
- 基本攻撃力はバニラの「鉄の剣」と同等。
- 耐久力はバニラの「ダイヤモンドの剣」と同等。
- 生物Mobを直接攻撃すると、短時間炎上させる。  
  (エンチャント：Break of Dawn の効果)
- 「生物Mobの種類(内部情報)がアンデッド」(以下、アンデッド)を１度でもDBで直接攻撃して、  
  そのアンデッドが生物(プレイヤー含む)の攻撃でトドメをさされると、一定確立で爆発(ダメージなし)する。  
  (エンチャント：Break of Dawn の効果)
- その爆発に巻き込まれたアンデッドは、最寄のプレイヤーから逃げようとする。  
  (動作は若干不安定ではある)
- 後述の「Meridia's Beacon Replica」(以下、メリ玉)と金床で修理可能。  
  (必要経験値は常時1。耐久値を最大まで修理)
- 修理回数に応じて、アンデッドに対して追加ダメージが発生。(修理回数 x ハート4分の1)  
  (エンチャント：Break of Dawn の効果)
- ワールドにはアイテムとして出現せず、独自の物体として顕現する。  
  その際、素手で右クリックする事で、入手する事が可能。
- 上記の状態では「奈落」に落ちない限り、破壊される事はない。

##■入手方法
###DB
1. プレイヤーによるダメージ(素手・剣・弓・ポーション等)でアンデッドにトドメをさす。
1. 10体倒すと、アンデッドの足元にチェストが出現する。(謙虚で無慈悲な強制設置)
1. 中に「メリ玉」が入っているので、バニラの「鉄の剣」とクラフトする。

###メリ玉
1. 初回時は10体のアンデッドにトドメをさす。
1. 2回目以降は100体のアンデッドにトドメをさす事で入手可能(つまり、110体討伐で2つ目入手)

##■その他
- アンデッドの討伐回数、及び、金床での修理回数は、プレイヤーデータなので、  
  DBをロストして新たに作成し直しても、情報は引き継がれる。(メリディア様はちゃんと見ていらっしゃる(脳内補正))
- 逃走するアンデッドには、クリーパーの帯電エフェクトのようなものが表示される。  
  (バニラのウィザーを除くアンデッドのみ対応)
